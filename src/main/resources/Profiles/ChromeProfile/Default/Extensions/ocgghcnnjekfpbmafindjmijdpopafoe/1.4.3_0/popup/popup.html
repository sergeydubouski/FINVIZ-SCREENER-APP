To automatically generate selectors with Ranorex Selocity, do the following:

  • After adding Ranorex Selocity, restart Chrome
  • Right-click an element on the web page and click 'Inspect' or open the Chrome DevTools by pressing F12
  • Find Ranorex Selocity on the right side of the Elements tab
  • Click the copy icon next to the generated selector in the list to copy it to the clipboard
